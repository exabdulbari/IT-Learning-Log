# Linux Basics

Basic Linux commands useful for IT support.

## 1. File and Directory Management
- List files: `ls`  
- Change directory: `cd /path/to/folder`  
- Make directory: `mkdir foldername`  
- Remove file: `rm filename`  
- Remove folder: `rm -r foldername`  

## 2. Searching and Viewing Files
- Search for text in files: `grep "keyword" filename`  
- View file content: `cat filename`  
- View file page by page: `less filename`  

## 3. Process and System Monitoring
- Check running processes: `top` or `ps aux`  
- Kill a process: `kill PID`  

## 4. Permissions
- Change permissions: `chmod 755 filename`  
- Change owner: `chown user:group filename`  

## 5. Network
- Check connectivity: `ping 8.8.8.8`  
- Display IP configuration: `ifconfig` or `ip addr`
